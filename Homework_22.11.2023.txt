1.Как скопировать на свой компьютер репозиторий из github.com, что должно быть сделано предварительно? Чтобы скопировать репозиторий с GitHub на свой компьютер, вам нужно склонировать репозиторий. Вот шаги, которые нужно выполнить:


Скопировать URL-адрес репозитория в GitHub.
Откройте терминал.
Создать директорю куда мы хотим  клонировать репозиторий.
Ввести команду git clone, а затем вставьте ранее скопированный URL-адре срепозитория.

2.Как создать git репозиторий из локального каталога на своем компьютере. Как его отправить на github.com? 


Откройте терминал.
Создать новую  рабочую директорию где мы хотим создать новый репозиторий.
Введите команду git init. 
Добавьте файлы в репозиторий - git add.
Зафиксируйте изменения - git commit.
Создайте пустой(без Readme ) репозиторий на GitHub.
Скопируйте URL-адрес удаленного репозитория.
Свяжите локальный репозиторий с удаленным репозиторием - git remote add origin <URL-адрес>.
Отправьте файлы -  git push.

3.Как получить на локальный компьютер изменения с github.com? 

Откройте терминал.
Выбрать директорию на место, где мы будем сохранять изменения.
Введите команду git pull.

4.Какую информацию нам дают команда git status, git log, git remote -v?

git status: Показывает состояние рабочего дерева и индекса. Он показывает, какие файлы были изменены, но еще не добавлены в индекс, какие файлы были добавлены в индекс, но еще не зафиксированы, и какие файлы не отслеживаются Git.
git log: Показывает историю коммитов.
git remote -v: Показывает список удаленных репозиториев и их URL-адреса.
