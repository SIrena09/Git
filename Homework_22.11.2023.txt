1.Как скопировать на свой компьютер репозиторий из github.com, что должно быть сделано предварительно? 
Чтобы скопировать репозиторий с GitHub на свой компьютер, нужно склонировать репозиторий. 
Скопировать URL-адрес репозитория в GitHub.
В терминале создать директорю куда мы хотим  клонировать репозиторий.
Ввести команду git clone, а затем вставить ранее скопированный URL-адре с репозитория

2.Как создать git репозиторий из локального каталога на своем компьютере. Как его отправить на github.com? 

В терминале создать новую  рабочую директорию где мы хотим создать новый репозиторий.
Ввести  команду git init. 
Добавить  файлы в репозиторий - git add.
Зафиксировать изменения - git commit.
Создать пустой(без Readme ) репозиторий на GitHub.
Скопироватье URL-адрес удаленного репозитория.
Связать локальный репозиторий с удаленным репозиторием - git remote add origin <URL-адрес>.
Отправить файлы -  git push.

3.Как получить на локальный компьютер изменения с github.com? 
В терминале выбрать директорию на место, где мы будем сохранять изменения.
Введите команду git pull.

4.Какую информацию нам дают команда git status, git log, git remote -v?
git status: Показывает состояние рабочего каталога и раздела проидексированых файлов. Также можно проверить индексацию изменений и увидеть файлы, которые не отслеживаются.
git log: Показывает историю коммитов.
git remote -v: Показывает список удаленных репозиториев и их URL-адреса.
